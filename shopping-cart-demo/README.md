# 使用方式


請直接修改 js 資料夾內的 products.js 內的陣列元素，要增減多少自行決定...


```
const products = [
    {
        id: '1',
        title: '產品一',
        price: 10,
        img: 'https://picsum.photos/id/999/1200/600',
        tags: ['生活用品', '工具'],
        isAvailable: true
    },
    {
        id: '2',
        title: '產品二',
        price: 60,
        img: 'https://picsum.photos/id/1070/1200/600',
        tags: ['藥妝'],
        isAvailable: true
    },
    ....
    // 以此類推
];
```

圖片都是透過 https://picsum.photos  上產生假圖 ....
